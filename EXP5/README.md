# 说明  
- **此EXP5是工科创II综合实验部分的代码。**  
该部分的要求分为两步，分别为基础部分和进阶部分。据我所知，基础部分的内容每年都一样，而进阶部分会有所改变。所以，请不要迷信这份代码。  
对于2023~2024-2学期，进阶部分内容为：
1. 发射端播放音频
2. 发射端音频送FM模块，接收端对应接收，直接播放。
3. 发射端音频送FM模块，接收端对应接收，先读入再播放（本组此部分用了测频率再用PWM输出的方法）。
4. 发射端音频送FM模块，接收端对应接收，并可录音一段时间，随后播放。
> 值得一提的是，虽然袁神一直强调进阶部分不要求做出来，可以放弃；但如果你听信了他的话，你的分数会非常难看。**建议每组都至少要实现一半以上的内容。**  

本目录下的两个子目录中，`5-6`为发射端的代码，`5-7`为接收端的代码。
